===== REQUIRED FIELDS =====

title: "Your Post Title"
  - The title of your blog post
  - Shows in post header and card previews

date: 2025-01-15 15:42:00 +/-0100
  - Publication date and time
  - Format: YYYY-MM-DD HH:MM:SS +/-TTTT
  - Can also use: 2025-01-15 (without time)


===== STANDARD JEKYLL FIELDS =====

published: true
  - Set to false to hide post from site (draft mode)
  - Default: true

permalink: /custom-url/
  - Override the default URL structure
  - Example: /my-custom-post-url/

layout: post
  - Override the default layout
  - Default: post (set in _config.yml)

excerpt: "A brief summary of the post"
  - Custom excerpt text
  - Auto-generated from content if omitted


===== CHIRPY THEME FIELDS =====

categories:
  - Blog
  - Learning
  - List of categories (hierarchical)
  - First level: Blog or Project
  - Second level: Learning, Tutorial, Deepdive, etc.

tags:
  - cpp
  - unreal-engine
  - visual-scripting
  - List of tags for filtering
  - Reference: _data/tags.yml for available tags

media_subpath: /assets/img/
  - Base path for images in the post
  - Images can use relative paths

image: thumbnail.jpg
  - Featured image for the post
  - Shows in post previews and social media
  - Can be full path or use media_subpath

pin: true
  - Pin post to top of blog listing
  - Default: false

toc: true
  - Show table of contents sidebar
  - Default: true (from _config.yml)

comments: true
  - Enable comment system for this post
  - Default: true (from _config.yml)

math: true
  - Enable MathJax for mathematical formulas
  - Default: false

mermaid: true
  - Enable Mermaid for diagrams
  - Default: false


===== CUSTOM FIELDS (Add Your Own) =====

author: "Boas-Bas van der Veen"
  - Override default author

reading-time: "10 min"
  - Custom reading time estimate

series: "Unreal Engine Tutorials"
  - Group related posts

part: 1
  - Part number in a series

last_modified_at: 2025-02-01
  - Last update date
  - Auto-generated by plugin if omitted

difficulty: "Intermediate"
  - Technical difficulty level

related-projects:
  - example-project
  - List of related project slugs
