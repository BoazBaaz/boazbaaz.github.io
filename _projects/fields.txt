===== REQUIRED FIELDS =====

title: "Your Project Title"
  - The title of your project
  - Shows in project header and card

date: 2025-01-15
  - Project completion or start date
  - Used for sorting projects
  - Format: YYYY-MM-DD


===== STANDARD JEKYLL FIELDS =====

published: true
  - Set to false to hide project from site (draft mode)
  - Default: true

permalink: /projects/custom-url/
  - Override the default URL structure
  - Default: /projects/title/

layout: project
  - Override the default layout
  - Default: project (set in _config.yml)


===== PROJECT DISPLAY FIELDS =====

description: "A brief description of this project"
  - Shows on project cards and project page header
  - 1-2 sentences recommended

thumbnail: /assets/img/project-name/thumbnail.jpg
  - Project card image
  - Recommended size: 600x400px
  - Shows in grid view and homepage

categories:
  - Project
  - Game
  - Category hierarchy
  - First level: Project
  - Second level: Game, Tools

tags:
  - cpp
  - unreal-engine
  - team
  - List of tags for filtering
  - Reference: _data/tags.yml for available tags

featured: true
  - Show on homepage in featured projects section
  - Default: false
  - Only 3 featured projects show on homepage


===== PROJECT METADATA FIELDS =====

start_date: 2024-10-01
  - Project start date
  - Format: YYYY-MM-DD

end_date: 2025-01-15
  - Project completion date
  - Format: YYYY-MM-DD
  - Can be omitted if still in progress

team-size: 6
  - Number of people on the project
  - Shows as badge with icon (ðŸ‘¥ 6)
  - Use 1 for solo projects

team:
  programmer: [bob, lorry]
  artist: [laysha, mike]
  designer: [alex]
  - Team members organized by role
  - Role names reference _data/roles.yml
  - Can use: programmer, artist, designer, producer, etc.
  - Can use leads: programmer-lead, artist-lead, designer-lead

duration: "3 months"
  - How long the project took
  - Examples: "2 weeks", "6 months", "1 year"

role: "Lead Programmer"
  - Your role on the project
  - Examples: "Programmer", "Technical Lead", "Solo Developer"

status: "Completed"
  - Current project status
  - Options: "Completed", "In Progress", "Archived", "On Hold"
